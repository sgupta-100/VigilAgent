# Vigilagent IDE Framework

## Vision
To provide a persistent, autonomous development and security ecosystem where the **"Get Shit Done" (GSD)** framework, **"Ralph Loop"**, and **"TestSprite"** oversee every project, task, and security assessment performed within the Antigravity workspace.

## Tech Stack
- **Frontend**: React, Tailwind (Modern Liquid Aesthetic)
- **Backend**: FastAPI (Python)
- **Intelligence**: GI5 (Generative Intelligence for 5th Gen Cyber Targets)
- **Quality**: TestSprite (Automated Quality Sentinel)
- **Forensics**: "The Archivist" PDF Reporting

## Core Mandates
1. **Sovereignty**: No data leaves the local environment.
2. **Proactivity**: Real-time threat detection and mitigation suggestions.
3. **Forensics**: High-fidelity evidence collection for every finding.
4. **Autonomous Quality**: Zero-prompt testing for every committed phase.
